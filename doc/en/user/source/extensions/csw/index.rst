Catalog Services for the Web (CSW)
==================================

This section discusses the Catalog Services for Web (CSW) community module for GeoServer. With this module, GeoServer supports retrieving and displaying items from the GeoServer catalog using the CSW service.

For more information on CSW, please refer to `OGC OpenGIS Implementation Specification 07-006r1 <http://www.opengeospatial.org/standards/specifications/catalog>`_ and the `OGC tutorial on CSW <http://www.ogcnetwork.net/node/630>`_.
